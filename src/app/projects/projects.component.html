Biglietteria teatro

Piattaforma che, successivamente all'accesso tramite login, offre un catalogo di spettacoli da poter prenotare. Si verrà reindirizzati ad una pagina per il pagamento (fittizio) e la conferma di acquisto del biglietto. La registrazione utente e i biglietti acquistati vengono memorizzati nel database della compagnia teatrale.

Source code: Biglietteria teatro repository

Tecnologie utilizzate:

  • Java
  • SpringBoot
  • Hibernate
  • MySQL
  • JavaScript
  • HTML
  • CSS
