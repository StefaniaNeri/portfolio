Tripz: travel booking platform

I've created a platform that offers a catalog of destinations available for booking. On the login page you can create an account as either an Admin or Base User.
Base users can select and book one or more trips, then go to the Cart to complete the booking. Admin users have additional privileges, including the ability to delete destinations and view the list of users.
The bookings, as well as the removal of trips and user registrations, update the database accordingly.

Source code: Tripz repository

Made with:

  • Java
  • SpringBoot
  • Hibernate
  • MySQL
  • JavaScript
  • HTML
